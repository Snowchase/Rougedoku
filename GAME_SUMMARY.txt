                              SUDOKLE
                         Game Summary & Keywords
================================================================================

GAME OVERVIEW
-------------
Sudokle is a modern mobile Sudoku puzzle game that combines classic gameplay
with social features, customization options, and a rewarding progression system.
Players solve daily puzzles across four difficulty levels while earning coins
to unlock themes, avatars, fonts, and music tracks.


CORE GAMEPLAY
-------------
- Classic 9x9 Sudoku puzzles with standard rules
- Daily puzzles generated with consistent seeding (same puzzle for all players)
- Four difficulty levels: Easy, Medium, Hard, Expert
- Tap-based input with intuitive number pad
- Hint system for assistance (coin cost)
- Timer tracking for each puzzle
- Error highlighting for incorrect entries


PROGRESSION SYSTEM
------------------
Coin Rewards:
  - Easy puzzle completion: 10 coins
  - Medium puzzle completion: 25 coins
  - Hard puzzle completion: 50 coins
  - Expert puzzle completion: 100 coins

Bonuses:
  - Time bonus (under 3 min): +50 coins
  - Time bonus (under 5 min): +30 coins
  - Time bonus (under 10 min): +15 coins
  - First clear bonus: +50 coins
  - Rewarded ad: +25 coins

Penalties:
  - Hint usage: -5 coins per hint


CUSTOMIZATION OPTIONS
---------------------
Themes:
  - Classic (default, free)
  - Dark (default, free)
  - Ocean Blue
  - Forest Green
  - Sunset Orange
  - Lavender Purple
  - Midnight

Number Fonts:
  - Default (free)
  - Bold
  - Rounded
  - Serif
  - Handwritten

Avatars (by category):
  - Animals
  - Nature
  - Food
  - Sports
  - Fantasy

Background Music:
  - Ambient tracks
  - Classical pieces
  - Electronic/Lo-fi beats


SOCIAL FEATURES
---------------
- User profiles with customizable avatars
- Friends list with add/remove functionality
- Friend codes for easy connection
- Global and friends leaderboards
- Stats comparison


SETTINGS & OPTIONS
------------------
- Board lock (prevent accidental input)
- Music toggle (on/off)
- Sound effects toggle (on/off)
- Haptic feedback
- Theme selection
- Dark mode support
